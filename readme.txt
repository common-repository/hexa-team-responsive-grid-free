A beautiful Responsive team grid system for Wordpress

== Description ==

Now you can create beautiful grids for your website team members  plugin, you can visit <a href="http://www.worldclassthemes.com" target="_blank">Worldclassthemes.com</a> for preview.


How to setup

https://www.youtube.com/watch?v=CXpvJ-TXtIU

Install the plugin in your wordpress website and open "Team -> Add New" to add team members, there are titles, descriptions, feature image, and some other required fields.

Now to setup on the page open "Hexa Team" menu and copy the the shortcode.

Create a new page and use the shortcode to place the Hexa Grid.

You can also change the background color of grids and also can limit the number of team members to be shown in the grid.


== Frequently Asked Questions ==

= Will my images fit with the grid system =

Yes, this plugin uses the large version of images you upload to the team custom post type.

= Can I style my grids =

Yes you can style such as color, font size, background colors etc.
